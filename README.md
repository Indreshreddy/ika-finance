## 📌 Project Overview

**IKA Finance** is a complete personal finance dashboard built as an internship submission. It provides users with an intuitive interface to manage their financial life – from recording daily transactions to visualizing spending patterns and setting budget goals. The application features role-based access (Admin/Viewer), persistent data storage, and beautiful data visualizations.

**Live Demo:** *Open `my-finance.html` in any modern browser*

## ✨ Key Features

### 📊 Dashboard
| Feature | Description |
|---------|-------------|
| Real-time Stats | Balance, Income, Expenses, Savings Rate |
| Cash Flow Trend | 6-month income vs expense line chart |
| Spending Breakdown | Current month expenses by category (donut chart) |
| Recent Activity | Last 5 transactions displayed instantly |

### 📝 Transaction Management
| Feature | Description |
|---------|-------------|
| CRUD Operations | Add, Edit, Delete transactions (Admin only) |
| Search & Filter | By description, type (income/expense), or category |
| CSV Export | Download complete transaction history |
| Pagination | 6 transactions per page for clean navigation |

### 💡 Smart Insights
| Feature | Description |
|---------|-------------|
| Top Spending | Identifies your highest expense category |
| Monthly Change | Compares spending between consecutive months |
| Smart Recommendation | AI-like insight based on spending behavior |
| Comparison Chart | Bar chart visualization of month-over-month data |

### 🎯 Budget Goals
| Feature | Description |
|---------|-------------|
| Category Limits | Set monthly budgets for Food, Housing, Transport, etc. |
| Progress Tracking | Visual progress bars show spending vs budget |
| One-Click Reset | Restore default budget limits |

### 👑 Role-Based Access Control
| Role | Permissions |
|------|-------------|
| **Admin** | Full access – Add/Edit/Delete transactions, Reset budgets |
| **Viewer** | Read-only – View all data but cannot modify |
